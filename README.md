# 🧩 Autism Screening ML Project

## Files in this project
- `app.py` — Main Streamlit Dashboard (run this!)
- `Autism_Child_data_in_excel.xlsx` — Dataset 1
- `autism_screening.csv` — Dataset 2
- `PROJECT_REPORT.md` — Full project report
- `README.md` — This file

## How to Run

### Step 1: Install Libraries
```bash
pip install streamlit scikit-learn pandas numpy matplotlib seaborn plotly openpyxl
```

### Step 2: Run the Dashboard
```bash
streamlit run app.py
```

### Step 3: Open Browser
Go to: http://localhost:8501
